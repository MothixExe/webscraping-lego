Bienvenue sur le Projet LEGO

Ce projet a été créé par Adrien Arthaud pour collecter automatiquement des données sur les sets LEGO. Explorez les différentes sections pour en savoir plus sur les thèmes, les sets et les crédits du projet.

Fonctionnalités

  • Collecte automatique des données des sets LEGO
  • Affichage des thèmes LEGO disponibles
  • Recherche et filtrage des sets LEGO
  • Affichage des détails de chaque set LEGO

L'objectif

Le but étant ici de créer un site web qui permet de visualiser les sets LEGO et d'en savoir plus sur eux. Le site web est créé avec Tailwind CSS et utilise l'API Brikse pour récupérer les données des sets LEGO. Des requêtes sont envoyées par javascript au serveur flask python qui lui-même envoie des requêtes à l'API Brikse pour récupérer les données des sets LEGO.

Attention, si vous lancez le serveur pour la première fois, il se peut que la base de données de sets soit vide, c'est normal.
Parcourez les différents thèmes afin de remplir la base de données au fur et à mesure (ceci évite une énorme phase d'appel sur l'API).
